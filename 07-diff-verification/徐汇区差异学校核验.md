## 1. 总体判断

- **AI 文件对徐汇四大公办的初中路径过度简化**：把“电脑派位/派位+对口”写成单一目标初中，例如“汇师/建襄 → 位育初”“向阳/高安一 → 徐汇中学”。官方表显示四大公办多数进入多个初中派位池，不应写成确定一对一。
- **人工表在多数徐汇对口组合上更接近官方表**，尤其是高安一、向阳、建襄、汇师、上海小学龙川、田林四小、高安一华展、光启等。
- **AI 新增的东二/康外路径存在硬错误**：东二并非对口市二初级；康外小学对口康健外国语实验中学，但官方表未把它列为九年一贯制直升校。
- **AI 新增师三实验路径基本正确**：官方表列入九年一贯制。
- **人工表“汇师中城 → 园南中学(北校)”需修正校名/校区**：2026 表写作“园南中学（罗城校区）”。
- **复旦附属徐汇实验、西南位育附属实验**均在 2026 表列为九年一贯制；但复旦附属徐汇实验另有“徐实验（部分）”进入园南/徐教院附中南分/徐汇南校区的表述，需按具体地块/年级过渡进一步确认。

## 3. 对 AI 文件 `01-schools/xuhui.md` 的建议修正

### 3.1 必须修正的硬错误

1. **四大公办初中路径不能写成单一目标初中**  
   建议改为：
   - 高安一（宛平）→ 南模初 / 徐汇中学 / 五十四 / 市四（按电脑派位/组合池）
   - 向阳 → 位育初级 / 市二初级（电脑派位块）
   - 建襄 → 位育初级 / 市二初级（电脑派位块）
   - 汇师（徐虹）→ 徐汇中学 / 市四

2. **删除或重写“东二 → 市二初级”**  
   2026 表未支持该路径；应改为“东二（部分）→ 市四 / 零陵 / 南洋初级”。

3. **康外不能写为九年一贯直升**  
   应改为“康外小学 → 康健外国语实验中学（对口入学，非九年一贯制序号）”。

4. **上海小学需拆本部与龙川**  
   - 本部 → 徐汇区教育学院附属实验中学
   - 龙川 → 徐汇区教育学院附属实验中学南部分校

5. **高安一需拆宛平与华展**  
   - 宛平校区进入南模/徐汇/五十四/市四池
   - 华展校区进入紫阳/位育附属徐汇科技实验池

### 3.2 可保留但需降确定性的判断

- “上海小学本部 → 徐教院附中价值高”可以保留，但应注明这是**对口关系强 + 社区评价上调**，不是官方梯队。
- “四大公办 Tier A”可以保留；搜狐等社区资料亦支持“建襄、高安一、向阳、汇师”为公办第一梯队：<https://www.sohu.com/a/880573507_121124009>。但初中端必须按官方派位池写风险。
- “师三实验九年一贯”可以保留。

## 5. 风险提示

- 徐汇初中升学的关键不是买房后直接锁定某所初中，而是**小学学籍满三年以上 + 对口/派位规则**。官方文件原文见：<https://www.xuhui.gov.cn/xxgk/portal/article/detail?id=8a4c0c069a97abda019d6747bc191bc6>。
- 本报告读取的是 2026 年公开表；2027 及以后可能调整。购房前应每年 4 月复核徐汇区教育局最新“公办小学招生划片范围”和“公办初中入学方式”。
- 本地宝/九考网为转载源，最终交易决策前应下载/核验徐汇区政府附件 PDF 原件，或致电徐汇区招生考试中心确认具体居委、门牌号、校区与年级过渡安排。
