## 0.5 户口与入学政策基础（2026 官方口径）

### 0.5.1 公办小学：人户一致 + 部分学校 3 年门槛 + 五年一户

依据 S1：静安区公办小学按地块划片招生，人户一致优先；根据报名人数与招生计划数，部分公办小学对口入学条件执行“适龄儿童及父母户籍在对口地块满 3 年、户主为儿童或父母、产权人/承租人为儿童或父母”；该类学校执行五年一户。

结论：

1. 静安是中心城区里对入户年限最敏感的区之一，热门校按 3 年硬卡规划更稳妥。
2. 用户 2026-04-28 起置换，若目标 2027 入学，已无法满足多数热门校 3 年门槛。
3. 非爆表小学仍可能可入，但需 2027 年 3-4 月复核当年预警和学校招生公告。

### 0.5.2 公办初中：全部对口、电脑派位、九年一贯制并存

依据 S5：静安初中入学方式包括九年一贯制直升、全部对口、电脑派位/集团派位。初中端通常不新增户籍年限门槛，硬门槛主要发生在小学端。

结论：

1. 南部老静安多见市西/育才等派位池，北部老闸北多见集团或全部对口。
2. “集团对口”不等于锁定某个校区，例如风华初集团不能写成确定风华东/西校。
3. 市北初地段生不能等同市北理选拔班。

### 1.1 静教附小学部

- **小学梯队**：A。
- **2026 初中路径**：静教附初中部（S5）。

- **方式**：九年一贯制。

### 1.2 一师附小 / 一师附小分校

- **小学梯队**：A。
- **2026 初中路径**：市西中学教育集团派位池（市西初、市西附、同济七一等，以 S5 当年表为准）。

- **方式**：电脑派位。
- **本版判断**：强小学 + 强派位池，但不是确定市西初。
- **风险**：小学端 3 年门槛风险高，且派位到七一/市西附也需接受。

### 1.3 闸北实验小学 / 明德校

- **小学梯队**：A-/B+。
- **2026 初中路径**：风华初级中学教育集团（S5）。
- **初中梯队**：A-/B+。
- **方式**：集团派位/对口池。
- **本版判断**：小学招生校区应拆本部/明德；初中端只写风华集团，不写锁定风华西校。
- **风险**：本部、明德、龙盛等校区差异及风华集团内校区/班型差异需继续跟踪。

### 1.4 大宁国际小学 / 大宁国际西校

- **初中梯队**：A-/B+。
- **本版判断**：大宁板块居住属性强，但初中端不锁定风华东校。
- **风险**：本部/西校小学端地块不同，集团同池不等于校区完全等价。

### 1.5 静安一中心

- **小学梯队**：A-/B+。
- **2026 初中路径**：育才初级相关派位池，含静安三中心、上外静小、协和双语培明、西康路三小等相关口径（S5/C5）。
- **初中梯队**：B+。
- **本版判断**：小学声誉强，但初中链路弱于静教附/风华/市西/市北，按 A-/强二梯队更稳妥。
- **风险**：派位池内存在民办/不同性质学校，需每年复核具体表述。

## 5. 风险与避坑

### 5.1 “集团/派位池”不能写成锁定某校区

闸北实验、大宁国际进入风华初级中学教育集团，不是买房锁定风华东校/西校。正式表必须用集团口径。

### 5.2 市北地段生≠市北理

市北理是全市选拔/班型分层结果，地段对口进入的是市北初平行班或相关校区。若买房目标是“市北理”，不能只依赖学区房。

### 5.3 静安热门小学 3 年风险高

静教附、一师附小、万航渡路、闸北实验、大宁国际、静安一中心、中山北路、彭浦一小等均应按 3 年风险上限规划，用户当前窗口要优先看非爆表替代路径。

### 5.4 “A 小学 + B 初中”的估值要降权

静安一中心小学端强，但初中端育才相关路径弱于静教附/风华/市西/市北；不应按双 A 估值。

### 5.5 南北静安风格不同

老静安偏愉快教育和派位池，老闸北偏抓学习和集团化；同为静安，房源、学校风格、家长预期差异很大。

## 8. 从 2026-04-28 起换房可进入的静安路径建议

### 8.1 优先候选

1. **市西小学 → 市西中学教育集团派位池**
   B+ 小学 + A/B+ 初中池，是当前窗口内相对可行的市西初路径，但仍需接受派位结果。

2. **永兴路二小 / 大宁路小学 / 和田路小学 → 市北初相关校区**
   低热小学 + 市北初路径，但必须按“地段平行班”估值，不等于市北理。

3. **闸北一中心 / 棋院实验 → 上外苏河湾实验**
   B 小学 + B/B+ 上升初中，确定性较高，适合稳妥路线。

### 8.2 稳妥候选

- **静安三中心 → 育才初级**：B + B+，确定性较好。
- **田家炳小学 → 闸北八中**：B + B，普通稳妥路径。
- **彭浦五小 → 彭浦初级**：可作为彭浦初替代入口，需核验当年预警。

### 8.3 谨慎候选

- **静教附 / 一师附小 / 万航渡路 / 闸北实验 / 大宁国际**：路径强，但小学端 3 年风险大，用户当前窗口基本不稳。
- **静安一中心 → 育才相关派位池**：小学端强，初中端不是 A 档头部。
- **中山北路小学 → 市北初西校**：路径强，但小学端可能 3 年硬卡，优先级低于永兴路二小。

### 8.4 一句话结论

静安整理后应从“名校标签”转向“可达性 + 真实初中路径”评估。**用户当前窗口不宜追静教附、一师附、闸北实验、大宁国际等 3 年热门校；更务实的静安路径是市西小学进市西集团池、永兴路/大宁路/和田路进市北相关校区、闸北一中心/棋院实验进苏河湾。**
