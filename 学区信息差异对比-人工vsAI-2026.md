# 学区信息差异对比：人工整理 vs AI整理（2026）

对比对象：
- 人工整理：`2026年上海小学+初中人工整理学区信息.md`
- AI整理：`01-schools/*.md`

结论日期：2026-05-09

## 1. 总体结论

人工表是“全市候选清单”，覆盖 13 个区、173 条小学-初中组合；AI整理目前只覆盖 7 个区：徐汇、黄浦、静安、普陀、杨浦、长宁、虹口。

因此第一类差异是**覆盖范围差异**：人工表中浦东、闵行、宝山、嘉定、青浦、松江没有对应 AI 区级文件。

第二类差异是**口径差异**：人工表偏“候选组合罗列”；AI文件偏“2026官方/社区交叉验证后的策略判断”，包含入户年限、预警校、派位/直升风险、用户 2026-04-28 起换房窗口可行性。

第三类差异是**具体对口/梯队差异**：集中在黄浦、静安、普陀、杨浦、虹口；这些地方人工表有若干组合或梯队判断与 AI文件冲突。

## 3. 分区差异

## 徐汇区

整体差异不算大，但 AI文件更强调“派位/本区学籍3年/预警校”风险。

主要差异：

1. **高安一、向阳、建襄、汇师的对口初中描述不同**
   - 人工表：多写成“南模初/徐汇中学/市四/五十四”“位育初级/市二”等混合池。
   - AI：拆成更明确的策略口径：
     - 汇师/建襄 → 位育初级
     - 向阳/高安一 → 徐汇中学
     - 但均为派位，不是确定一对一。

2. **上海小学的重要性 AI 显著上调**
   - 人工表：上海小学总部/龙川均列为“二梯队+一梯队”。
   - AI：上海小学 → 徐教院附中被列为“徐汇双学区最硬通货”，但同时提示预警校3年、本区学籍3年硬门槛。

3. **人工表中部分学校 AI矩阵未纳入或降优先级**
   - 人工表有：汇师中城、上海小学龙川、田林四小、复旦附属徐汇实验、西位实验、高安一华展、光启小学。
   - AI矩阵没有作为核心双学区列入，需判断是资料缺口还是AI认为策略价值较低。

4. **AI新增/明确了人工表未强调的路径**
   - 东二 → 市二初级
   - 康外小学 → 康外中学
   - 师三实验 → 师三实验初中

## 黄浦区

黄浦差异较大，主要是“学校梯队”和“对口初中”冲突。

1. **曹光彪小学梯队冲突**
   - 人工：曹光彪小学 = 二梯队，组合“二梯队+一梯队”。
   - AI：曹光彪小学 = A，格致初级 = A，双A，但热门校有中等风险。

2. **上外黄浦梯队冲突**
   - 人工：上外黄浦 = 一梯队，大同初级 = 一梯队。
   - AI：上外黄浦外国语 = B，大同初级 = A；组合仍强，但小学档次下调。

3. **卢湾二中心梯队/组合冲突**
   - 人工：卢湾二中心 = 一梯队，向明初级 = 二梯队。
   - AI：卢湾二中心 = B，向明初级 = A；组合方向反过来。

4. **师专附小对口冲突**
   - 人工：师专附小 → 大境初级。
   - AI：师专附小 → 敬业初级。
   - 这是硬冲突，需要优先查官方2026黄浦初中对口表。

5. **徽宁路三小对口冲突**
   - 人工：徽宁路三小 → 市南中学，普初。
   - AI：徽宁路三小 → 大同初级，A。
   - 差异非常大，需官方核验。

6. **卢湾一中心对口冲突**
   - 人工：卢湾一中心 → 上海交通大学附属黄浦实验中学。
   - AI：卢湾一中心 → 兴业中学，B，且标为“不匹配”。

7. **AI新增强组合，人工缺失**
   - 七色花小学 → 向明初级
   - 北京东路小学 → 格致初级
   - 复兴东路三小 → 格致初级

## 静安区

静安主要差异在“梯队上调/下调”和“是否可行”。AI文件明显更重视3年硬卡、派位池、地段班风险。

1. **静安一中心梯队冲突**
   - 人工：静安一中心 = 二梯队，育才初级/培明 = 二梯队。
   - AI：静安一中心 = A，但3年硬卡，且派位到育才/协和培明，风险中等。

2. **大宁国际小学梯队冲突**
   - 人工：大宁国际小学 = 二梯队。
   - AI：大宁国际小学 = A，派位风华系，但3年硬卡。

3. **闸北一中心组合评级冲突**
   - 人工：闸北一中心 → 苏河湾实验，标“二梯队+一梯队”。
   - AI：闸北一中心 = B，苏河湾实验 = B，非爆表、可行，但不是一梯队初中。

4. **中山北路/永兴路到市北的风险被AI大幅细化**
   - 人工：中山北路 = 二梯队+一梯队，永兴路二小 = 普小+一梯队。
   - AI：二者到市北初级西校，但强调“地段生平行班，非市北理”，且中山北路3年硬卡，永兴路较可行。

5. **人工有但AI未列核心/或名称不同**
   - 闸北实验明德校区、大宁国际西校、闸北二中心、民办彭浦实验小学。

6. **AI新增可行替代路径**
   - 静安三中心 → 育才初
   - 棋院实验 → 上外苏河湾实验
   - 田家炳小学 → 闸八中
   - 西康路三小 → 育才初
   - 大宁路小学/和田路小学 → 市北初级北校

建议优先核验以下“硬冲突”，因为会直接影响买房/换房判断：

1. 黄浦：师专附小 → 大境初级 / 敬业初级
2. 黄浦：徽宁路三小 → 市南中学 / 大同初级
3. 黄浦：卢湾一中心 → 交大黄浦实验中学 / 兴业中学
4. 虹口：虹口四中心 → 虹口实验 / 鲁迅初级
5. 虹口：虹口三中心 → 鲁迅初级 / 曲阳二中
6. 虹口：红旗小学 → 丰镇中学 / 复兴初级
7. 普陀：AI所称“2026官方数据修正版”与人工表中普陀若干一贯制/高预期路径的取舍
8. 杨浦：打一是否仍对应鞍山初级；许五→鞍山是否应纳入人工表
9. 静安：市北相关路径是否应标注“地段生平行班，非市北理”
10. 长宁：人工表需补充“号块派位、非确定对口”的风险提示

## 5. 建议处理方式

1. 先不要直接合并人工表和AI表。
2. 先对“硬冲突”查各区2026官方小学/初中对口表。
3. 对每条路径增加三个字段：
   - 对口确定性：确定对口 / 派位 / 九年一贯 / 统筹或未公示
   - 入户门槛：无明确 / 建议年限 / 预警校 / 3年硬卡
   - 用户可行性：可行 / 可冲 / 高风险 / 已错过
4. 人工表保留为“广覆盖候选库”；AI表作为“策略研究与风险判断库”。最终应产出一张标准化总表，而不是用其中一个覆盖另一个。
