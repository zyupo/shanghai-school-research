## 一、黄浦区公办初中梯队总览

黄浦区是上海面积最小的区，但市重点高中资源全市领先（9所市重点高中）。中考竞争相对宽松，全区市重点率约 40%+，被称为"上海中考性价比之王"。

**梯队分类（社区多源交叉，非官方排名）：**

| 梯队 | 学校 | 核心竞争力 | 数据来源 |
|---|---|---|---|
| **一梯队** | 格致初级中学 | 黄浦北部核心强校，2025到校名额119个，理科竞赛强 | 多源一致 |
| **一梯队** | 大同初级中学 | 2025到校名额78个，与大同高中直升资源深厚 | 多源一致 |
| **一梯队** | 向明初级中学 | 原卢湾片核心强校 | 多源一致 |
| **二梯队** | 卢湾中学 | 2025到校名额55个，均分稳定，小道消息"均分曾排第一" | vsxue/知乎 |
| **二梯队** | 敬业初级中学 | 稳定B+，出口不差 | 多源一致 |
| **二梯队** | 大境初级中学（上外附属） | B/B+ | 多源一致 |

> 民办强校补充：民办立达、永昌是黄浦区生源最强的初中，但为摇号入学，非学区房可锁定路径，不在本文讨论范围。

## 二、一梯队初中详细分析

---

### 🥇 1. 格致初级中学

**学校基本情况**
- 地址：人民广场板块，云南南路附近
- 与格致高中一脉相承，直升资源明确
- 2025届参加中考人数约250人，到校名额119个（市重点率约47%）
- 理科、数学竞赛类强

**入学条件（2025/2026官方口径）**
- 入学方式：**小学对口初中**（全部对口）
- 入学路径：在对口小学完成六年小学后，对口直升格致初级中学
- 基础条件：**本区户籍 + 实际居住地与户籍地一致（人户一致）**
- 入户年限：曹光彪小学要求人户一致且实际居住满约3年（社区口径；官方未写固定年限，超额按入户时间排序）
- 五年一户：同一地址五年内只享有一次同校入学机会（双胞胎/多胞胎除外）
- 跨区情况：非黄浦户籍或外区小学学籍，全区统筹安排（不能锁定格致初级）

| **北京东路小学**（北京东路566号） | 虎丘、北京、无锡、龙泉、厦门、贵州、牛庄、云中、东风、宁波、山北 | 11个居委，地段相对宽 |

## 四、入学资格核心要求汇总

根据2025/2026黄浦区官方政策，公办初中入学要求如下：

| 条件 | 说明 |
|---|---|
| **人户一致** | 儿童户籍地与实际居住地必须一致 |
| **小学对口** | 必须在对应小学就读完6年；跨区学生需统筹安排，无法锁定目标初中 |
| **五年一户** | 同一地址五年内只享有一次同校对口入学机会（双胞胎/多胞胎除外） |
| **超额排序** | 热门小学报名超额时，按入户籍时间长短排序 |
| **入户年限** | 官方未设统一硬门槛，但热门校（如曹光彪、上外黄浦）实际需约2-3年提前入户才有保障 |
| **非沪籍** | 非沪籍或外区户籍，由区统筹安排，不能对口一梯队初中 |

## 六、买房前必须核验清单

- [ ] 产证地址是否完整落入目标小学 2027年对口居委范围
- [ ] 确认"人户一致"：户籍地与实际居住地必须同一地址
- [ ] "五年一户"核验：该地址5年内是否有其他孩子用过对口入学名额
- [ ] 前户主/原户主户口是否可按约定迁出
- [ ] 入户时间是否满足热门校的排序要求（2027入学 → 最好2024-2025年前已入户）
- [ ] 向明初级中学搬迁新校区后，新地段划片是否已更新
- [ ] 2027年3-4月招生文件发布后再次复核对口关系
