# 人工整理 vs AI整理差异学校核验

核验日期：2026-05-09

本目录只处理用户要求范围内的 7 个区：徐汇、黄浦、静安、普陀、杨浦、长宁、虹口。浦东、闵行、宝山、嘉定、青浦、松江按要求忽略，不做比对。

核验原则：

1. 对口关系优先采用区教育局/区政府发布的 2026 招生文件。
2. 本地宝、上观、网易等转载官方表格仅作可读交叉验证，最终仍应以官方原件为准。
3. 小红书、微信公众号、知乎、B站等社区内容只作为梯队/热度/风险参考，不能替代官方对口关系。
4. 梯队不是官方口径，文档中一律保留“待确认/社区口径/风险提示”。

## 最重要的修正方向

- 徐汇：AI 对四大公办初中派位池过度简化；东二→市二初级、康外九年一贯为错误口径。
- 黄浦：师专附小、徽宁路三小、卢湾一中心等硬对口采用人工/官方口径。
- 静安：采用集团/派位池口径；市北路径必须标注“地段生 ≠ 市北理”。
- 普陀：采用 2026 官方修正版；确定对口和集团/属地电脑派位必须分开写。
- 杨浦：打一不再写鞍山；许五→鞍山、五角场→同济初级、上理工附小→上理工附初应补入。
- 长宁：采用号块电脑派位机制；AI 中部分号块错误需按官方号块表修正。
- 虹口：四中心、三中心、红旗、广灵路等硬冲突采用人工/官方口径；上外附小→上外附中东校待确认。
